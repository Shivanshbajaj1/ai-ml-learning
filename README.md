# AI / ML Learning 🤖

A structured, beginner-friendly path through AI/ML fundamentals using Python.

## How to use this repo

Each folder is one topic, in suggested order. Inside each folder:
- `README.md` — concept explanation
- numbered `.py` files — runnable code (`python 01_xxx.py`)

Work through folders **01 → 08** in order.

## Roadmap

| # | Topic | Folder |
|---|---|---|
| 1 | Python Foundations (NumPy, Pandas) | `Numpy/` 'Pandas/'|
| 2 | Data Visualization | `Matplotlib/` |
| 3 | Data Preprocessing | `data_preprocessing/` |
| 4 | Supervised Learning | `Supervised_learning/` |
| 5 | Unsupervised Learning | `Unsupervised_learning/` |
| 6 | Model Evaluation | `Model_evaluation/` |
| 7 | Neural Networks | `Neural_networks/` |
| 8 | Mini Projects | `Projects/` |

## Requirements

```bash
pip install numpy pandas matplotlib scikit-learn
```

(`07_neural_networks` uses only NumPy — no deep learning framework required to
understand the fundamentals first.)

## How to learn from this repo

1. Read the topic README for the *why*.
2. Run the file, change the numbers/data, see what breaks.
3. Re-implement the core function (e.g. gradient descent step) without
   looking, from memory.
4. Move to `08_mini_projects/` once you've been through 01–07, and apply
   everything to a small project of your own.
