# 02. Data Visualization

## Why visualize before modeling?

You can't trust a model on data you haven't looked at. Plots reveal outliers,
skewed distributions, missing patterns, and relationships between features —
all things summary statistics alone can hide (see "Anscombe's Quartet" if curious).

## Plot types and when to use them

| Plot | Use case |
|---|---|
| Line plot | Trends over time/sequence |
| Histogram | Distribution / spread of a single variable |
| Scatter plot | Relationship between two numeric variables |
| Bar chart | Comparing categories |
| Box plot | Spread + outliers at a glance |
